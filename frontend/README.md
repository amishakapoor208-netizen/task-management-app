# Frontend (React + Vite + Tailwind)

Quick start:

1. Install dependencies:

   cd frontend
   npm install

2. Start dev server:

   npm run dev

Environment:
- You can set VITE_API_URL to point to the backend (default http://localhost:4000)

Pages:
- /register
- /login
- /tasks (protected)

Notes:
- JWT is stored in localStorage for this scaffold.
